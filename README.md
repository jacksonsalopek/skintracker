# SkinTracker

## Description

SkinTracker is a NestJS & Commander-based CLI that allows users to quickly view CSGO skin prices. Users can add skins to a tracking list, and then query the list across multiple CSGO markets (i.e. BitSkins). Instructions are provided below.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
